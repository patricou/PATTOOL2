Caches TV & radio (mémoire serveur)

  • Chaînes TV : playlist par pays, ~60 min. Rechargée au prochain accès après expiration. Les pays sans playlist (404) sont mémorisés vides pour éviter de les redemander.
  • Programmes TV (EPG) : guide par pays, ~180 min. Source iptv-epg.org — tous les pays du catalogue sont essayés au refresh, mais seuls ceux qui ont un fichier EPG sont remplis. Même dans un pays couvert, seules les chaînes avec un id EPG (tvg-id) ont un programme.
  • Radio : pays, stations (par pays) et genres mondiaux, ~60 min.

Mises à jour automatiques

  • ~90 s après le démarrage du serveur : gros refresh complet (toutes les playlists TV, EPG tous pays, radio tous pays).
  • Chaque nuit à 04:00 : même refresh complet.
  • À 07:00, 17:00 et 20:00 : rafraîchissement EPG + radio de tous les pays.

Bouton « Rafraîchir les caches » : même refresh complet, lancé à la demande. Un message s’affiche quand c’est terminé. Un seul refresh à la fois.

Le redémarrage du backend vide ces caches. Ils ne sont pas stockés dans le navigateur (sauf le total TV pour le badge).

TV & radio caches (server memory)

  • TV channels: per-country playlist, ~60 min. Reloaded on the next access after expiry. Countries with no playlist (404) are cached empty so they are not re-fetched repeatedly.
  • TV programmes (EPG): per-country guide, ~180 min. Source iptv-epg.org — every catalog country is tried on refresh, but only countries with an EPG file are filled. Even then, only channels with an EPG id (tvg-id) get programme data.
  • Radio: countries, stations (per country) and worldwide genres, ~60 min.

Automatic updates

  • ~90 s after server start: full refresh (all TV playlists, EPG all countries, radio all countries).
  • Every night at 04:00: the same full refresh.
  • At 07:00, 17:00 and 20:00: EPG + radio refresh for all countries.

“Refresh caches” button: the same full refresh, on demand. A message appears when finished. Only one refresh runs at a time.

Restarting the backend clears these caches. They are not stored in the browser (except the TV total for the badge).
