Qu'est-ce que WhatsPat ?

WhatsPat remplace l'ancien chat Firebase. Les messages sont stockés dans MongoDB et mis à jour en direct via WebSocket/STOMP. L'accès est sécurisé par votre compte Keycloak.

Types de discussions

GénéraleSalon global — page /results
Par activitéUne discussion liée à chaque événement (discussionId)
Par groupe d'amisDiscussion de groupe depuis la page Amis

Comment y accéder

  • Menu principal → entrée WhatsPat / Résultats → /results
  • Pied de carte d'une activité complète (vue grille)
  • Fiche détail d'une activité
  • Mur de photos — bouton discussion sur chaque bloc
  • Groupes d'amis
En « vue compacte » (titre seul) sur la liste des activités, le pied de carte — donc le bouton discussion — n'est pas affiché.

Fonctions

  • Texte en temps réel, indicateur de connexion WebSocket
  • Images et vidéos en pièce jointe dans les messages
  • Page /results : liste des discussions (générale, activités, groupes) avec aperçu du dernier message
  • Modale de discussion redimensionnable, défilement automatique
  • Statistiques des discussions (admin) depuis la page Système

Utilisation

  1. Ouvrez une discussion (depuis /results ou une activité).
  2. Saisissez votre message en bas ; Entrée pour envoyer.
  3. Joignez une image ou une vidéo si le bouton pièce jointe est disponible.
  4. Les autres participants connectés voient les messages en direct.
  5. Sur une activité sans discussion : la première ouverture crée automatiquement le fil WhatsPat.
Les commentaires sur une activité (bouton commentaires) sont distincts de WhatsPat : fil classique, pas de temps réel WebSocket.

What is WhatsPat?

WhatsPat replaced the legacy Firebase chat. Messages live in MongoDB and update live via WebSocket/STOMP. Access is tied to your Keycloak account.

Discussion types

GeneralGlobal room — /results
Per activityOne thread per event (discussionId)
Friend groupGroup chat from Friends page

How to open it

  • Main menu → WhatsPat / Results → /results
  • Footer of a full activity card (grid view)
  • Activity detail page
  • Photo wall — discussion button on each block
  • Friend groups
In title-only compact view on the activity list, the card footer (and discussion button) is hidden.

Features

  • Live text, WebSocket connection indicator
  • Image and video attachments
  • /results: discussion list with last message preview
  • Resizable discussion modal, auto-scroll
  • Discussion statistics (admin) on System page

How to use it

  1. Open a thread (from /results or an activity).
  2. Type at the bottom; Enter to send.
  3. Attach image or video when the attach button is available.
  4. Other connected users see messages in real time.
  5. First open on an activity creates the WhatsPat thread if missing.
Activity comments are separate from WhatsPat (classic thread, not WebSocket).
