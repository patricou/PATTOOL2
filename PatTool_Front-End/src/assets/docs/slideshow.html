Ouvrir le diaporama

  • Sur une carte activité : bouton photos / diaporama
  • Sur le mur de photos : clic sur une miniature
  • Sur la fiche détail d'une activité : boutons photos ou fichiers image
  • Sources possibles : fichiers joints, liens photo, dossiers disque serveur (PHOTOFROMFS)

Fonctions utiles

Zoom

Clic pour zoomer, molette, clic droit + zone, double-clic pour dézoomer.

Carte GPS

Affiche la position EXIF ; bascule carte / photo avec E.

EXIF

Panneau métadonnées : I.

Diaporama auto

Lecture automatique : S.

Téléchargement

Bouton DL — image visible (y compris si zoomée).

Vidéos

Lecture intégrée ; Espace lecture/pause.

Opening the slideshow

  • On an activity card: photos / slideshow button
  • On the photo wall: click a thumbnail
  • On activity details: photo buttons or image files
  • Sources: uploads, photo links, server disk folders (PHOTOFROMFS)

Main features

Zoom

Click to zoom, mouse wheel, right-drag area, double-click to zoom out.

GPS map

EXIF location; toggle map/photo with E.

EXIF

Metadata panel: I.

Auto slideshow

Auto advance: S.

Download

DL button — visible area (including zoom).

Videos

Inline playback; Space play/pause.
