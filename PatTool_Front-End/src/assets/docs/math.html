Loto (FDJ)

Route : #/tools/loto

  • Tableau des tirages stockés en MongoDB (archive oct. 2008 → présent)
  • Filtres par date, tri, recherche dans le tableau
  • Graphiques de fréquence des numéros et du numéro chance
  • Export JSON (format IA) depuis une modale dédiée
  • Analyse IA : envoie un prompt + JSON des derniers tirages à l'assistant (nouvelle conversation)
Admin : import d'une plage de mois depuis l'archive FDJ, correction de date de tirage, rafraîchissement du tableau.

EuroMillions (Millionnaire)

Route : #/tools/euromillions

  • Historique des tirages, filtres, tri par date / combinaison / gain / code
  • Méthodes statistiques (χ², entropie, récurrence d'écart, corrélation des sommes, Monte Carlo) — cartes avec étoiles indicatives
  • Graphiques : fréquences, évolution mensuelle, comparaison de méthodes
  • Synchronisation des tirages (admin) et paramètre de date minimale pour l'IA
  • Analyse IA : prompt + enveloppe JSON (stats + méthodes cochées) → assistant avec Claude Opus 4.8
Les étoiles et analyses sont indicatives — pas une prédiction garantie des tirages.
