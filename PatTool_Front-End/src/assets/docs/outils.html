Éditeur PDF

Route : #/tools/pdf-converter — connexion requise.

Rédigez un document riche (texte + images), enregistrez la mise en page sur le serveur, puis exportez un PDF identique ou partagez-le sur WhatsApp.

Documents enregistrés

  • Liste déroulante de vos documents MongoDB (un par utilisateur)
  • Nouveau document — brouillon vierge
  • Supprimer — suppression définitive du document cloud

Éditeur (Quill)

  • Barre d'outils : titres, gras, italique, listes, liens, couleurs…
  • Insérer une image — depuis fichiers ; glisser-déposer dans le texte
  • Redimensionner — cliquer l'image, tirer les coins
  • Supprimer l'image sélectionnée — ou touche Suppr / Retour arrière
  • Nom de fichier PDF (sans extension .pdf, max 180 caractères)

Actions

BoutonEffet
EnregistrerSauvegarde HTML + nom + option pied de page dans MongoDB
Générer le PDFTélécharge le PDF (génération côté serveur via l'API assistant/export)
WhatsAppGénère le PDF puis ouvre le partage (mobile : menu système ; légende copiée si besoin)
Tout effacerVide l'éditeur local (sans supprimer le document cloud sélectionné)

Option pied de page : nom, date, auteur (le numéro de page reste affiché).

Enregistrez avant de changer de document dans la liste pour ne pas perdre les modifications non sauvegardées.

Tableur Calc (.ods)

Route : #/tools/ods-editor — connexion requise.

Éditeur de feuilles de calcul OpenDocument dans le navigateur (compatible LibreOffice / OpenOffice Calc).

Ouvrir et créer

  • Ouvrir un fichier .ods — depuis votre disque ; couleurs et formatage d'origine conservés à l'affichage
  • Nouveau classeur — grille vide
  • Classeurs enregistrés sur le serveur — liste déroulante comme pour le PDF

Grille (AG Grid)

  • Édition au clic dans les cellules
  • Ajouter une ligne / colonne
  • Nouvelle feuille — désactivée si le formatage d'origine doit être préservé (structure fixe)
  • Onglets pour basculer entre feuilles
  • Curseur taille du texte (8–20 px, mémorisé localement)
  • Nom du fichier et nom de la feuille active éditables
  • Badge Modifications non enregistrées tant que le classeur est « sale »

Enregistrement

BoutonEffet
Enregistrer sur le serveurBlob .ods + métadonnées dans MongoDB (rechargement ultérieur)
Télécharger .odsFichier OpenDocument sur votre machine
SupprimerEfface le classeur cloud sélectionné

Lancement depuis une activité

Sur le mur de photos ou une fiche activité, un lien tableur ODS peut ouvrir l'éditeur avec le fichier déjà chargé (paramètre d'URL / service de lancement).

PDF editor

Route: #/tools/pdf-converter — login required.

Rich text + images editor (Quill). Save layout to MongoDB, generate matching PDF, or share via WhatsApp.

  • Cloud document list, new, delete
  • Insert/drag/resize images in text
  • Save, Generate PDF, WhatsApp, optional footer (name, date, author)

Calc spreadsheet (.ods)

Route: #/tools/ods-editor — OpenDocument editor (AG Grid), open local .ods or cloud workbooks.

  • Edit cells, add rows/columns/sheets (sheet add limited when preserving original formatting)
  • Save to server (MongoDB) or Download .ods
  • Font size slider, multi-sheet tabs, unsaved badge
  • Can open from activity ODS links on photo wall or detail page
