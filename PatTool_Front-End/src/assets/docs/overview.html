Qu'est-ce que PatTool ?

PatTool est une application web privée pour organiser des activités (sport, sorties, famille, voyages…) et partager photos, fichiers et discussions avec un cercle d'amis authentifié via Keycloak.

Mur de photos

Page d'accueil par défaut — timeline des photos par activité.

Activités

Création, inscription, fichiers, visibilité fine.

WhatsPat

Messagerie temps réel (MongoDB + WebSocket).

Organisation

Agenda, listes de tâches, rappels e-mail.

Assistant IA

Chat latéral : OpenAI, Claude, Gemini, Mistral. Recherche web, vision, images, historique MongoDB, exports PDF/WhatsApp. Guide complet →

Outils

IoT, météo, Loto, PDF, tableur ODS, globe 3D…

Rôles et accès

Keycloak attribue des rôles qui débloquent certaines fonctions :

RôleExemples
Utilisateur connectéActivités, amis, WhatsPat, assistant, mur de photos
FileSystemLiens photos sur disque serveur (PHOTOFROMFS)
IotMenu Maison : domotique, caméras, scan réseau
AdminAdministration liens, paramètres serveur, stats système

La visibilité des activités (public, privé, amis, groupe) est gérée par l'auteur, indépendamment des rôles admin.

Premiers pas

  1. Connectez-vous via Keycloak.
  2. Le mur de photos (/photos) s'affiche par défaut.
  3. Consultez vos activités dans Activités → Liste (/even).
  4. Ouvrez l'assistant (baguette à droite) pour poser une question.
  5. Menu Outils : upload global de fichiers, partage, documentation, PDF, tableur ODS.

Technologies (résumé)

  • Frontend : Angular 21, Bootstrap 5, Capacitor 8 (mobile)
  • Backend : Spring Boot 3.3, Java 21, MongoDB
  • Temps réel : WebSocket/STOMP pour WhatsPat ; SSE pour le flux d'activités
  • Fichiers : GridFS et disque serveur (photos PHOTOFROMFS, rôle FileSystem)
Note : l'ancien chat Firebase a été remplacé par WhatsPat. Il n'est plus utilisé.

What is PatTool?

PatTool is a private web app to organize activities (sports, outings, family, travel…) and share photos, files and discussions with friends, authenticated via Keycloak.

Photo wall

Default home — photo timeline per activity.

Activities

Create, join, attach files, visibility rules.

WhatsPat

Real-time messaging (MongoDB + WebSocket).

Organization

Calendar, todo lists, e-mail reminders.

AI assistant

Side panel: OpenAI, Claude, Gemini, Mistral. Web search, vision, images, MongoDB history, PDF/WhatsApp export. Full guide →

Tools

IoT, weather, lottery, PDF, ODS, 3D globe…

Roles and access

RoleExamples
Logged-in userActivities, friends, WhatsPat, assistant, photo wall

IotHome menu: automation, cameras, network scan

Getting started

  1. Sign in via Keycloak.
  2. Photo wall (/photos) is the default home.
  3. Browse activities at /even.
  4. Open the assistant (wand button on the right).
  5. Tools menu: global upload, share, documentation, PDF, ODS editor.

Technology (summary)

  • Frontend: Angular 21, Bootstrap 5, Capacitor 8 (mobile)
  • Backend: Spring Boot 3.3, Java 21, MongoDB
  • Real-time: WebSocket/STOMP (WhatsPat); SSE (activity stream)
  • Files: GridFS and server disk (PHOTOFROMFS, FileSystem role)
Note: legacy Firebase chat was replaced by WhatsPat.
