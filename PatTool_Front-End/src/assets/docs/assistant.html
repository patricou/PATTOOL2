Accès et prérequis

  • Connexion obligatoire : l'assistant n'est visible que si vous êtes authentifié via Keycloak.
  • Bouton flottant (baguette magique) sur le bord droit de l'écran — présent sur presque toutes les pages.
  • Un point de notification sur le bouton indique qu'une réponse est prête si le panneau était fermé pendant la génération.
  • Cliquez en dehors du panneau (zone grisée) ou sur × pour fermer.
Sécurité : vos messages ne partent jamais directement du navigateur vers OpenAI, Anthropic, Google ou Mistral. Tout transite par le serveur PatTool, qui détient les clés API.

Fournisseur, modèle et guide

Les quatre fournisseurs

OpenAI

GPT-5.x, GPT-4.x, série « o » (raisonnement). Seul fournisseur avec MCP et génération d'images OpenAI.

Anthropic

Claude (Fable 5, Opus 4.8, Sonnet 4.6, Haiku 4.5). Recommandé pour les activités (lancement auto).

Google Gemini

Texte, long contexte, vision et génération d'images Gemini. Utilisé pour les photos et les actualités.

Mistral AI

Large, Small, Pixtral (vision), Codestral (code), Ministral (léger).

Choisir un modèle

  1. Sélectionnez le fournisseur dans la liste déroulante.
  2. Choisissez un modèle dans la deuxième liste, ou « Modèle personnalisé… » et saisissez l'id API exact.
  3. Ouvrez le Guide modèles pour des tableaux par tâche (rédaction, code, vision, économique, recherche web…) et cliquez Sélectionner ce modèle.

Votre choix fournisseur/modèle peut être mémorisé côté serveur pour votre compte (préférence de routage).

Guide modèles (bouton ⓘ)

Ouvre une fenêtre modale plein écran avec :

  • Matrice outils — même tableau que ci-dessous (recherche web, images, MCP par fournisseur)
  • Classement par tâche — or / argent / bronze : quel modèle privilégier pour rédaction, code, vision, économique, etc.
  • Fiches par fournisseur — id API, forces, limites ; bouton Sélectionner ce modèle pour l'appliquer immédiatement

Ordre de priorité du routage

  1. Préférence enregistrée sur le serveur (MongoDB) pour votre compte
  2. Choix de la session navigateur (jusqu'à fermeture de l'onglet)
  3. Valeurs par défaut du serveur PatTool

Un lancement depuis une activité ou un article peut écraser temporairement le fournisseur/modèle pour cette ouverture uniquement.

Envoyer un message

Saisie

  • Entrée — envoyer
  • Maj + Entrée — nouvelle ligne
  • Bouton agrandir — éditeur de brouillon plein écran avec recherche dans le texte
  • Bouton copier — copie le brouillon dans le presse-papiers
  • Bouton × sur le champ — vide le brouillon sans effacer la conversation affichée

Joindre une image (vision)

  • Bouton Image : appareil photo ou galerie/fichiers
  • Formats : JPEG, PNG, GIF, WebP — max 8 Mo
  • Avec une image jointe, saisissez obligatoirement votre question avant d'envoyer
  • L'aperçu image dans le fil peut ne pas être conservé après rechargement de l'historique

Format des réponses

Les réponses sont affichées en Markdown (titres, listes, tableaux, blocs de code). Les images générées par l'IA apparaissent dans la bulle ; vous pouvez les agrandir, les copier ou les rattacher à une activité.

Pendant la génération

  • Indicateur sablier avec compteur de durée en millisecondes
  • Champ de saisie, envoi, sélecteurs et historique désactivés jusqu'à la fin de la réponse
  • Si vous fermez le panneau pendant l'attente, le point de notification sur le bouton flottant signale que la réponse est prête
  • Pas d'annulation en cours de route : attendez la fin ou rechargez la page en dernier recours

Historique des conversations

  • Bouton Historique en tête du panneau
  • Enregistrement automatique sur le serveur (MongoDB) après chaque réponse, avec léger délai
  • Filtre de recherche dans la liste des conversations
  • Actions : Ouvrir, copier la première question, Effacer (suppression définitive côté serveur)
  • La gomme sous le champ efface l'affichage local pour démarrer un nouveau fil ; l'historique serveur reste accessible

Partage WhatsApp et export PDF

WhatsApp (conversation entière)

  • Choisir : question + réponse, ou réponse seulement
  • Sur mobile : menu de partage système → WhatsApp
  • Texte très long : peut être tronqué ; complément optionnel dans le modal

Export PDF

Télécharge un PDF de la conversation avec le détail fournisseur/modèle par tour. Nécessite au moins un échange dans le fil.

Crédits et facturation

  • Bouton graphique en tête : ouvre la console facturation du fournisseur sélectionné (URL configurée côté serveur)
  • OpenAI : PatTool peut afficher le solde prépayé (USD) si l'API billing est configurée
  • Anthropic, Gemini, Mistral : pas de solde dans PatTool — lien vers la console du fournisseur

Conseils d'utilisation

  • Activité / rédaction : Claude Sonnet ou Opus, ou GPT-5.x — lancez depuis la carte activité pour un routage adapté
  • Photo / diaporama : Gemini Flash Image avec l'image déjà jointe
  • Actualité : Gemini + case Recherche web
  • Code : Codestral, o4-mini, ou Claude Opus
  • Rapide et pas cher : Haiku, GPT-4o mini, Mistral Small
  • Illustration : cocher Génération d'images + OpenAI ou Gemini
  • Utilisez la gomme entre deux sujets différents pour éviter de mélanger les contextes
  • Consultez le Guide modèles intégré pour le tableau comparatif par type de tâche

Access and requirements

  • Login required: the assistant is only available when authenticated via Keycloak.
  • Floating button (magic wand) on the right edge — on almost every page.
  • A notification dot appears when a reply is ready while the panel was closed.
  • Click the dimmed backdrop or × to close.
Security: messages never go directly from the browser to OpenAI, Anthropic, Google or Mistral. Everything goes through the PatTool server, which holds API keys.

Provider, model and guide

Four providers

OpenAI

GPT-5.x, GPT-4.x, “o” series (reasoning). Only provider with MCP and OpenAI image generation.

Anthropic

Claude (Fable 5, Opus 4.8, Sonnet 4.6, Haiku 4.5). Default for activity launches.

Google Gemini

Text, long context, vision and Gemini images. Used for photos and news.

Mistral AI

Large, Small, Pixtral (vision), Codestral (code), Ministral (lightweight).

Choosing a model

  1. Pick a provider in the first dropdown.
  2. Pick a model in the second list, or “Custom model…” and enter the exact API id.
  3. Open the Model guide for task-based tables and click Select this model.

Your provider/model choice may be persisted on the server for your account.

Model guide (ⓘ button)

Opens a modal with: tools matrix, gold/silver/bronze ranking per task type, and per-provider model sheets with Select this model.

Routing priority

  1. Server preference (MongoDB) for your account
  2. Browser session (until tab close)
  3. PatTool server defaults

Launches from activities or news may override provider/model for that opening only.

“Tools” checkboxes (below the input)

Optional — slower replies and higher API usage. Only compatible options are shown for the active provider.

ToolOpenAIGeminiAnthropicMistral
Web searchYesYesYesYes
Image generationYesYesNoNo
MCPYes*NoNoNo

* MCP requires a server configured on the backend (openai.mcp.*). Checkbox is disabled if none is set up.

  • Web search: news, weather, stocks, recent facts — ideal with Gemini or Claude + checkbox on
  • Image generation: ask for an illustration in the message; Gemini may switch to a dedicated image model server-side

Sending a message

  • Enter — send
  • Shift + Enter — new line
  • Expand — full-screen draft editor with in-text search
  • Image button — camera or gallery (JPEG, PNG, GIF, WebP, max 8 MB)
  • With an attached image, you must type your question before sending

Reply format

Replies are rendered as Markdown (headings, lists, tables, code blocks). Generated images appear in the bubble with copy / attach / WhatsApp actions.

While generating

  • Hourglass indicator with elapsed milliseconds
  • Input, send, selectors and history disabled until the reply completes
  • Closing the panel while waiting shows a notification dot on the floating button when ready
  • No in-flight cancel — wait for completion or reload as last resort

Conversation history

  • History button in the header
  • Automatic server save (MongoDB) after each reply
  • Search filter, open, copy first question, delete permanently
  • The eraser clears the local display only; server history remains
