Créer et gérer une activité

  1. Créer : menu Activités → nouvelle activité, ou /neweven
  2. Types : VTT, ski, rando, vin, famille, etc. (~20 types)
  3. Visibilité : public, privé, amis, ou groupe d'amis
  4. Participants : inscription / désinscription sur la carte
  5. Fichiers : glisser-déposer ou bouton upload (images, PDF, vidéos, traces GPX…)
  6. Liens : sites web, photos en ligne, dossiers serveur (PHOTOFROMFS — rôle FileSystem)
  7. Commentaires : fil de commentaires classiques (≠ WhatsPat)

Mur de photos

  • Grille masonry par activité, chargement progressif
  • Filtre par nom, visibilité, groupe d'amis
  • Notes 👍/👎 par activité (barre compacte en haut de bloc)
  • Upload direct depuis le mur, traces GPS, lien todo list
  • Bouton Assistant (baguette) pour interroger l'IA sur l'activité
  • Traces GPX/KML : tableau distance, dénivelé, lien vers visionneuse carte
  • Section « Un jour comme aujourd'hui » : photos d'années précédentes

PHOTOFROMFS et fichiers serveur

Type de lien Photo depuis le système de fichiers : photos stockées sur le disque du serveur (pas en base). Nécessite le rôle FileSystem.

  • Ajout à la création ou modification d'activité (sélecteur de dossier)
  • Visible dans le diaporama et le mur de photos comme les autres images
  • Badges de liens sur le mur : PDF, tableur ODS, site web, carte, etc.

Fiche détail et liens transverses

  • /details-evenement/:id — vue complète : participants, commentaires, discussion, fichiers, trace GPS
  • Lien vers todo list associée (si définie)
  • Partage WhatsApp de l'activité ou du mur filtré
  • Diaporama, visionneuse de trace, upload depuis la fiche ou le mur

Three separate controls (easy to mix up):

ControlEffect
Grid iconFull cards (element-evenement) — default
List iconHorizontal list with action bar
Compact view toggleSimplified cards: image + title bar (fewer footer buttons)
WhatsPat on a full card is in the card footer. In title-only compact view, that footer is hidden.

Create and manage an activity

  1. Create: Activities menu → new, or /neweven
  2. Types: MTB, ski, hiking, wine, family, etc.
  3. Visibility: public, private, friends, or friend group
  4. Participants: join / leave on the card
  5. Files: drag-and-drop or upload (images, PDF, video, GPX…)
  6. Links: websites, online photos, server folders (PHOTOFROMFS — FileSystem role)
  7. Comments: classic comment thread (≠ WhatsPat)

Photo wall

  • Masonry grid per activity, progressive loading
  • Filter by name, visibility, friend group
  • 👍/👎 ratings per activity (compact bar on each block)
  • Direct upload, GPS tracks, todo list link
  • Assistant (wand) button to query AI about the activity
  • GPX/KML tracks: distance, elevation, map viewer link
  • « On this day » banner: photos from previous years

PHOTOFROMFS and server files

Photo from file system link type: photos on server disk (not GridFS). Requires FileSystem role.

  • Add when creating or editing an activity (folder picker)
  • Shown in slideshow and photo wall like other images
  • Link badges on the wall: PDF, ODS spreadsheet, website, map, etc.

Detail page and cross-links

  • /details-evenement/:id — full view: participants, comments, discussion, files, GPS trace
  • Linked todo list when configured
  • WhatsApp share for activity or filtered photo wall
  • Slideshow, trace viewer, upload from detail or wall
